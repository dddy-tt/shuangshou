## 二、传感器

| 模块 | 型号/规格 | 数量 | 状态 | 位置 | 说明 |
|------|----------|:---:|:---:|------|------|
| 柔性弯曲传感器 | 4.5 寸 (11.4cm) | 8 | ✅ | 食指/中指/无名指/小指 | 10KΩ 分压，电阻 10~100KΩ |
| 柔性弯曲传感器 | 2.2 寸 | 2 | ✅ | 拇指 | 拇指短关节专用 |
| **JY61P IMU** | 维特智能 JY61P | **2** | ⚠️ **已有** | 左右手背各一 | ⚠️ 需在 PC 上位机中切成 **I2C 模式**（默认串口模式） |
| MAX30102 | 心率血氧模块 | 1 | ✅ | 左手食指指尖 | I2C (PC6-7), 软 I2C 400kHz |

**JY61P 配置注意**：
- 出厂默认串口模式 → 必须用维特智能 PC 上位机切成 I2C 模式（只切一次，固化保存）
- I2C 地址 0x50（7 位写地址），本代码用 8 位写地址 0xA0
- 采样率 200Hz，卡尔曼滤波内置

## 三、通信模块

| 模块 | 型号 | 数量 | 状态 | 接口 | 说明 |
|------|------|:---:|:---:|------|------|
| 蓝牙 | **JDY-33** | 1 | ⚠️ **需新购** | USART3 (PC10 TX, PC11 RX) | **必须 JDY-33 不是 JDY-31**！JDY-31 不支持 BLE，Web Bluetooth 连不上 |
| WiFi | **ESP-01S** (ESP8266) | 1 | ✅ | PE0 软 UART (TX only) | AT 固件，用 AT+MQTTPUB 发 MQTT |

**蓝牙⚠️重要**：
- 之前买的 JDY-31 不能用（经典蓝牙 SPP，不支持 BLE）
- JDY-33 是 BLE 4.2/5.0 协议，支持 Web Bluetooth API
- 引脚兼容，直接拔插替换
- 淘宝搜 "JDY-33 蓝牙模块" ~¥12

**ESP-01S 配网（必须先做）**：
```
AT+CWMODE=1
AT+CWJAP="WiFi名称","WiFi密码"
AT+MQTTUSERCFG=0,1,"","","",0,0,""
AT+MQTTCONN=0,"broker.emqx.io",1883,0
```
用 USB-TTL 连 ESP-01S 手动发一次，配置固化后就不需要了。

## 七、引脚分配总表

### ADC（10 路手指弯曲）
```
ADC1_IN0  PA0  右手拇指     ADC2_IN5  PA5  左手拇指
ADC1_IN1  PA1  右手食指     ADC2_IN6  PA6  左手食指
ADC1_IN2  PA2  右手中指     ADC2_IN7  PA7  左手中指
ADC1_IN3  PA3  右手无名指   ADC2_IN8  PB0  左手无名指
ADC1_IN4  PA4  右手小指     ADC2_IN9  PB1  左手小指
```

### 软 I2C（三路 GPIO 翻转，400kHz）
```
SoftI2C_1  PB6(SCL) PB7(SDA)  右手 JY61P
SoftI2C_2  PB8(SCL) PB9(SDA)  左手 JY61P
SoftI2C_3  PC6(SCL) PC7(SDA)  MAX30102
```

### UART
```
USART1  PA9(TX)  PA10(RX)   DFPlayer Mini（DMA 非阻塞发送）
USART3  PC10(TX) PC11(RX)   JDY-33 蓝牙
```

### 软 UART
```
PE0  TX only   ESP-01S RX（115200bps DWT 精确定时）
```

### TIM
```
TIM3          1ms 系统调度时钟（PSC=83, ARR=999 → 1kHz）
TIM4 CH1 PE2  右手振动 PWM
TIM4 CH2 PE3  右手振动 PWM
TIM4 CH3 PE4  左手振动 PWM
TIM4 CH4 PE5  左手振动 PWM
```
